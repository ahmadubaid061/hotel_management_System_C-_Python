# 🏨 Hotel Management System

A simple **Hotel Management System** built in both **Python** and **C++** using Object-Oriented Programming (OOP) concepts.

---

## 📁 Project Structure

```
hotel_management_System_C-_Python/
│
├── using_python/
│   └── code.py          # Python implementation
│
├── using_C++/
│   └── code.cpp         # C++ implementation (coming soon)
│
└── README.md
```

---

## 🧠 OOP Concepts Used

- **Class & Object** — `Room` class to represent each hotel room
- **Encapsulation** — room data is managed through a single class
- **Constructor** — `__init__` to initialize room attributes

---

## 🏠 Room Types

| Type | Description | Price Per Night |
|------|-------------|-----------------|
| Class A | Luxury style room | Rs. 100,000 |
| Class B | Room with 2 beds | Rs. 50,000 |
| Class C | Single bed room | Rs. 25,000 |

- Total Rooms: **30**
  - Class A: 5 rooms (Room 1–5)
  - Class B: 10 rooms (Room 6–15)
  - Class C: 15 rooms (Room 16–30)

---

## ⚙️ Features

- ✅ View all rooms
- ✅ View occupied rooms
- ✅ View available rooms
- ✅ Book a room (by room type)
- ✅ Vacate a room (by guest name + room ID)
- ✅ Exit

---

## 🖥️ How to Run (Python)

Make sure you have Python 3.10+ installed (required for `match/case` syntax).

```bash
# Clone the repository
git clone https://github.com/ahmadubaid061/hotel_management_System_C-_Python.git

# Navigate to the python folder
cd hotel_management_System_C-_Python/using_python

# Run the program
python code.py
```
